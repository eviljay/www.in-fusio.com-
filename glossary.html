G L O S S A R Y

3G

Shorthand for "Third Generation" wide-band services.

Third Generation services are expected to use microwave frequencies allowing speeds of up to 2Mbit/s, using packet-switched networks for voice and data transmission.

UMTS will be such a service. The world is predominantly "Second Generation" with GSM at the moment. Most operators will go through "2.5G", referring to GPRS.

Bluetooth

A global initiative initiated by Ericsson, IBM, Intel, Nokia and Toshiba to set a standard for wireless communication between any two devices. Initial applications will be between mobile handsets, PDAs and portable PCs. It will also enable phones to become "pocket" devices with wireless earpieces. It uses low-powered 2.45GHz microwave short-range radio links in the unregulated Instrumentation Scientific and Medical range.

Circuit Switching

Historic means of enabling a phone call by establishing a circuit, or link, throughout the network between two devices and the beginning of a call, and closing it at the end. The circuit stays open regardless of the amount of data being transferred.

The advantage of circuit switching is that it is perfectly "real-time" as the circuit is not shared by other users, and thus ideally suited for a telephone conversation, videoconferencing and multimedia. The disadvantage is that it is extremely costly to maintain a link between two points, making it less well suited for permanent data connections, with bursty traffic such as e-mail. This issue is solved by Packet Switching.

EDGE
(cf GPRS)

Enhaced Data rates for Global Evolution changes radio coding in a way that allows GPRS to carry upto 384 kbit/s

EPOC

Operating System under development by Symbian, based on Psion’s developments, for multimedia terminals.

GPRS

General Packet Radio Service is an enhancement to GSM’s initial design to allow packet switching over a radio interface. Demos will take place through-out 2000. Large numbers of commercial handsets not expected until mid 2001.

GSM

Global System for Mobile communications is now the worlds main digital cellular standard, covering over half of all mobile users worldwide, with 362 operators in 132 countries, and 285 million users by the end of Q1 2000.
See http://www.gsmworld.com/membership/mem_stats.html

HDML

Handheld Device Markup Language is non-normalised pre-WAP language developed by Phone.com.

HSCSD

High Speed Circuit Switched Data will be the last evolution before GPRS for establishing data links over a circuit switched mobile network. This should enable data speeds of upto 57.6kbit/s.

Few operators are expected to invest in HSCSD, moving straight to GPRS or and UMTS.

I-mode

I-mode is NTT DoCoMo’s mobile internet service that has taken the world by storm, surprising everyone by the speed of its growth, reaching 5.8 million subscribers by April 2000.
See http://www.eurotechnology.com/imode/index.html

MexE

Mobile Execution Environment. Likely to be based on Java technology, MeXe will enable WAP (Wireless Application Protocol)-enabled devices to offer a wider range of features with greater security and flexibility. MeXe will also enable greater control of telephony features than is possible with WAP alone.

Java

Java is computer programming language developed by Sun that is designed to operate on any computer, interpreting the program as it runs. It has proved extremely popular in websites. The lesser power of mobile phones, however, means that it is expected to be excessively slow for full game animation on these devices.

Packet Switching

A transmission technique optimised for bursty data transfer as it only uses network or airtime resources when data is actually being sent. It works by cutting a message into packets that each carry the destination’s address, allowing them to be mixed with other packets, or to travel over different circuits, and then be correctly reassembled at the other end.

SIM

The Subscriber Identity Module takes the form of a chip etched onto a card that is inserted into a mobile phone to allow the network to identify the subscriber. It also stores the algorithms that encrypt speech. With the Sim Toolkit, applications can be downloaded into the card and run on the phone.

SIM Toolkit (STK)

The toolkit allows the operator or service provider to personalise a phone’s menu structure and improve the user interface by tailoring it to specific applications.

Smart phones

An in-between denomination for devices that are a merged phone and organiser, often incorporating full keyboards.

SMS

Short Message Service enables GSM phone users to exchange data messages of up to 160 characters. It uses the signalling level of GSM networks, rather than the voice circuit switched network. Despite user constraints, SMS-messaging between phone users has exploded in all countries in which it is available.

Symbian

A Psion initiative joined by Ericsson, Nokia and Motorola designed to make phones run better applications. Symbian’s main product is EPOC, a dedicated operating system, that competes with Windows CE. The consortium is trying to make the technology a standard for mobile computing and phones. It is licensed by Philipps and Sony.
See http://www.symbian.com/news/latest_pr.html

UMTS

Universal Mobile Telecommunications System is the European initiative for 3G. Initially planned for 2001, it seems unlikely that any real commercial applications with large numbers of handsets will be available before early 2003.

WAP

Wireless Application Protocol is the global open standardised version of HDML which allows on-line services to be used via a small-screen mobile phone. All phone manufacturers have signed up to include WAP browsers in their phones, and most major operators have declared their intent to provide WAP services.

The general view is that most internet access in Europe will soon come from mobile phones. Cap Gemini America forecast in April 2000 that up to 72 million WAP phones could be sold in America this year, compared to 45 million PCs.

Some say, however, that given the limited number of handsets, the acronym actually means: Where Are the Phones?
See http://www.wapforum.com/
