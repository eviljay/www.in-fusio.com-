In-Fusio rolls out API deal

Motorola and In-Fusio are to collaborate in a joint venture to develop a new games application programming interface (API) for mobile phone.

The advanced API is being designed to assist game developers create more compelling games for Java technology-enabled mobile communication products and other wireless devices, that deliver fast action and highly attractive graphics.
"The market opportunity for mobile games is tremendous. Games provide a rigorous environment for testing software architectures, and they quickly raise market awareness for the entire value chain of mobile communication products," said Anne-Marie Larkin, vice president and director of Motorola's wireless software applications and services. "A unique standardised games API will give developers a consistent framework from which to write compelling, fun programs that consumers can download to Java technology enabled handsets."
